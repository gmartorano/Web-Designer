A história do HTML

O básico do HTML ao HTML5

O que é o "HTML"?

HTML é uma abreviação da expressão inglesa "HyperText Markup Language", que significa Linguagem de Marcação de Hipertexto. É uma linguagem de marcação utilizada para produzir páginas da web, documentos HTML que podem ser interpretados por navegadores.

A criação do HTML

Em 1980, um físico britânico chamado Tim Berners-Lee, propôs um projeto baseado no conceito de hipertexto para facilitar o compartilhamento e a atualização de informações entre os pesquisadores, nascia ai o HTML. Sua solução combinada com a internet pública ganhou a atenção mundial.

Etiquetas (tags)

Todo documento HTML possui tags, palavras entre < e >; essas tags são comandos de formatação da linguagem.
Uma tag pode ou não possuir atributos e valores. Os atributos são utilizados para modificar o resultado padrão do elemento.

Edição de arquivos HTML

Os documentos em HTML são arquivos de texto simples que podem ser criados e edeitados em qualquer editor de textos comum, como o Bloco de Notas do Windows, ou o TextEdit, do Mac OS.

Para facilitar a produção de documentos, no mercado existem editores de HTML específicos, com recurso sofisticados, que facilitam a realização de tarefas repetitivas, inserção de objetos, elaboração de tabelas e outros rescursos, basicamente dividem-se em dois tipos:

Editores de texto fonte
Inserem automaticamente as etiquetas, orientando a inserção de atributos e marcações, como o Notepad++
Editores WYSIWYG
Oferencem um ambiente de edição visual, com um "esboço" do resultado final, como o Dreamweaver

O Web HyperText Application Techonology Working Group (WHATWG) iniciou o trabalho do novo padrão HTML em 2004, enquanto o World Wide Web Consortium (W3C) estava se concentrando no desevolvimento do XHTML 2.0. Em 2009, o W3C decidiu que a equipe de desevolvimento do XHTML 2.0 deveria parar seus trabalhos, e assim, descontinuar o padrão. Desta forma o W3C e o WHATWG passaram a trabalhar juntos no desevolvimento do HTML5.

O projeto do HTML5 foi bem recebido pelos desevolvedores Web até então, e tornou-se tema na mídia em abril de 2010 depois que o CEO da Apple Inc., Steve Jobs emitiu uma carta pública intimulada "Reflexões sobre o Adobe Flash", onde ele conclupia que o desenvolvimento do HTML5 tornaria o Adobe Flash não mais necessário, tanto parar assistir vídeo ou mesmo exibir qualquer conteúdo da web.

A popularidade do HTML5

De acordo com um relatório divulgado em 30 de setembro de 2011, 34% dos 100 melhoresWeb sites do mundo estavam usando HTML5. A adaptação foi liberada por mecanismos de buscas e redes sociais. Em 20 de outubro de 2011, o Facebook anunciou o lançamento do Centro de Recursos HTML, dando aos desenvolvedores ferramentas para construir, testar e implementar aplicações para o Facebook.

Logo do HTML5

Em 18 de janeiro de 2011, a W3C introduziu um logo para representar o uso do HTML5. Diferente de outros emblemas anteriormente disponibilizandos pela W3C, este não implicam uma validação ou o uso correto da linguagem. Desde o di 1º de abril, este logotipo é oficial.
Segundo a W3C, o logotipo "Representa o HTML5, a pedra angular para aplicações web modernas".
